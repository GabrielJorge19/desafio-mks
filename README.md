# Descrição dos scripts (ex: npm run [script])

- start - Inicia o servidor de desenvolvimento.
- build - Faz o build do typescript.
- api - Inicia a API.
- server - Inicia o servidor estático.
- buildCompleto - Executa os comandos "build", "api" e "server".
